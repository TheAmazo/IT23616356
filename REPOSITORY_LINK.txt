Repository URL: https://github.com/TheAmazo/IT23616356

Setup Instructions
==================

1. Clone the repository:
   git clone https://github.com/TheAmazo/IT23616356.git
   cd IT23616356

2. Install dependencies:
   npm install

3. Install Playwright browsers:
   npx playwright install --with-deps

4. Run tests:
   npm test

5. View results:
   npm run report

Full documentation is available in README.md in the repository.
